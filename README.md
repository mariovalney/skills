# Claudinho

### AI Best-Practices Audit Skill

This skill turns any repository into an AI-native codebase by doing a structured research phase first, then producing an approved plan, and only then implementing changes.
It never modifies the tech stack (only the instruction layer).

When to use this skill:

- User wants to add or improve Claude instructions in a repository.
- User wants to audit an existing repo for AI-readiness.
- User asks "how do I make Claude Code work better here?"
- User pastes a task similar to the one that defines this skill (research two reference repos, then plan).
